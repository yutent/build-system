# Build System
> 🔥 类似sublime的Build System功能, 方便快速调试一些代码片断或想法。
>> 目前只支持脚本语言的简单编译调试, 因为vscode原则上只是一个编辑器, 我并不希望把它变成一个IDE, 所以暂时并不打算支持那些需要复杂的编译环境及配置的语言。
